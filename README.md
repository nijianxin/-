1. 项目本质（What this project actually is）  本项目并非游戏 AI、自动化脚本或视觉点击工具。  它是一个研究平台，用于回答一个更基础的问题：  计算机是否可以在不依赖具体界面结构的情况下执行任务？  我们尝试构建一种 界面无关（Interface-Independent）的可执行行为语言， 并研究如何将该语言映射到任意图形化程序（GUI）。  换句话说：  人类使用软件时的过程是：  目标 → 理解功能 → 操作界面  而计算机自动化通常是：  识别像素 → 点击坐标  本项目试图建立第三条路径：  目标 → 行为语义 → 映射到界面  GUI 在这里不再是“被识别的对象”， 而是“可实现行为的执行环境”。
